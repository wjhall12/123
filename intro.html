    How can we change the style of text on a web page?

    So far we have only made web pages where we control the content and structure, such as which parts of the pages are headings or paragraphs. We've been using HTML as the language to specify the content and structure of the pages. While HTML allows us some control over how the page looks, it doesn't give developers much control over the specific look and style of each element. To do that, we need a language to express style.

    What is CSS?

    CSS - Cascading Style Sheets; a language used to describe how HTML elements should be styled

    CSS Selector - the part of a CSS rule-set that defines which HTML elements the style should be applied to


    Students should understand that the structure of the page is there to organize the information in a logical way, but doesn't tell the computer how to display it. So, for example, the structure could include different headings and paragraphs, but would say nothing about the color of text or how big it is displayed. The style of the page is the specifics of its appearance. Without a particular style, each web browser would decide how to display different web page elements on their own.


    Styles are important to allow web developers to decide exactly how a web page looks on the screen. Because styles are separate from structure and content, web developers can change the style of an entire page very easily, without having to make any changes to the structure and content of the web page. This means that it's very easy for web pages to have an individual look and feel that is unified across the entire page.



    Introduced Code

    color: value; - With CSS, colors can be specified in different ways: By color names, RGB values, and hexadecimal values. All modern browsers support 140 color names and thousands of HEX Values


    Color Names



    HEX Color Values


    font-family: value; - In CSS, we use the font-family property to specify the font of a text. In CSS there are five generic font families:

    Serif fonts have a small stroke at the edges of each letter. They create a sense of formality and elegance.
    Sans-serif fonts have clean lines (no small strokes attached). They create a modern and minimalistic look.
    Monospace fonts - here all the letters have the same fixed width. They create a mechanical look.
    Cursive fonts imitate human handwriting.
    Fantasy fonts are decorative/playful fonts.


    font-size: value;; - The font-size property sets the size of the text. Being able to manage the text size is important in web design. However, you should not use font size adjustments to make paragraphs look like headings, or headings look like paragraphs.

    text-align: value; - The text-align property is used to set the horizontal alignment of a text. A text can be left or right aligned, centered, or justified.

    text-decoration: value; - The text-decoration-line property is used to add a decoration line to text.


    Selecting by Tag Name

    We use CSS rules to select elements on a web page so that we can then style those elements. The way we tell our CSS rule which HTML elements to style is by using selectors. The element selector selects HTML elements based on their tag names. Each HTML element

    and any other HTML element can be selected with CSS by using the tag name without the angle brackets (< and >). Instead, in CSS you use curly braces { } . For example, you can select all of the paragraphs tags in your webpage by using the element selector p. Here's a CSS rule that changes the color of each paragraph on a web page: p { color: blue; }

        Display Elements

        Block, Inline, Inline-block, None

        Display Elements


        A display:block element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element. A block-level element always takes up the full width available (stretches out to the left and right as far as it can). Two commonly used block elements are: p and div. Which means paragraph and headings are default to display block.


        An Inline element allows content to stay on the same line, as opposed to block which makes content start on a new line. Also the inline element removes any width or height elements you've made.


        An Inline-block element allows content to stay on the same line, and allows you to set the width and height on the element. With inline-block, you can change the the top/bottom margins and padding, but with inline you can't. One common use for inline-block is to display list items like the nav bar horizontally instead of vertically.


        If you say display None you will simply hide the element from showing in the browser
